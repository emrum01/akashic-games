# 広告画像の素材クレジット / 生成方法

## 写真広告 `ads/adphoto-*.jpg`（主役・全9種）

現実の胡散臭いWeb広告（記事風・写真広告）に使う写真は、**codex CLI の組み込み `image_gen` ツール
（OpenAI gpt-image-2）で新規生成**したものです（外部素材の取り込みではありません）。生成主体は
codex で、ローカルで `ads/_gen/` に元写真（1536×1024）を出力しました。

- codex の画像生成可否の検証結果: **可能**。`codex exec` は組み込み `image_gen` スキル（gpt-image-2）を
  持ち、写真的なラスタ画像を直接生成できる（検証で 1536×1024 の写真JPEGを生成確認）。ただし
  1セッションで6枚をまとめて生成させると停止したため、**1セッション1枚**に分けて生成した。
- 被写体（すべて「安っぽい胡散臭い広告のストック写真」風に指示）: 腹部＋メジャー（belly）/
  白いサプリ瓶＋黄カプセル（supplement）/ 手に持った札束（cash）/ 驚いて指さす人物（face）/
  濁った健康飲料（food）/ 白衣＋聴診器で瓶を持つ人物（doctor）/ 皿の健康食（meal）/
  警告画面のスマホ（phone）/ 郊外の一軒家（house）。
- テキスト・ロゴ・実在ブランドは含めない指示。生成物は本ゲーム用途に使用。

### クリックベイト加工（`ads/_gen/*.jpg` → `ads/adphoto-*.jpg`）

元写真に ImageMagick 7 で以下を合成し、現実の詐欺広告らしい体裁に加工した
（スクリプトは制作時の使い捨て。手順のみ記録）:

- 462×308（3:2）へリサイズ＆センタークロップ。
- 上部に**黄色い煽り帯**（`#ffd400`／赤い縁）＋赤い極太見出し（ヒラギノ角ゴ W8、例「衝撃の-12kg」
  「医師が警告」「月30万円」）。
- 被写体に**赤い楕円（丸囲み）＋赤い矢印**を重ね、視線を誘導。
- 左下に小さく目立たない「PR」タグ（半透明黒地＋白字）。
- **JPEG品質52**で再圧縮し、わざと軽く劣化させて安っぽさを出す（1枚13〜24KB）。

## 旧素材（不使用）

以前の `ad-*.svg`（codex生成のSVGアイコン）と `adimg-*.jpg`（CC0写真合成のクリックベイト）は、
「現実のWeb広告の再現」方針への転換に伴い**不使用**（game.js から参照されていない）。
参照は全て `adphoto-*.jpg`（写真広告）に置換済み。
