# assets/（後工程で配置）

以下を置くと質感が上がるが、**未配置でも `index.html` はCSSフォールバック（多重グラデーション背景・CSSクリップ文字）で完全に動作・見栄えが成立する**。

| ファイル | 用途 | 参照箇所 |
|----------|------|----------|
| `bg_thread.jpg`（1080×1920 縦） | 暗いオカルト板の背景テクスチャ | `#water-bg` の `--bg-image`（未配置ならグラデのみ） |
| `sumi_ink.png`（1024×1024 透過） | 墨のにじみマスク（任意） | 文字の質感付け |
| `ogp.jpg`（1200×630） | X共有カード | `<meta property="og:image">` |

画像を置いたら、`bg_thread.jpg` は `#water-bg` の `--bg-image` にCSSで差し込むだけで反映される（onerror不要。存在しなければ `none` のまま）。
