# まっすぐ読むな｜動物園脱出

閉園後の動物園から脱出するモバイルファースト RPG 風アドベンチャー Web 謎。
静的 HTML/CSS/JS のみ。ビルド不要。公開想定 URL: https://games.escape-safari.com/zoo-escape/

## ファイル構成

- `index.html` … ゲーム本体（HTML + inline CSS + inline JS）
- `analytics.js` … GA4 計測ラッパー（`window.AkashicAnalytics`）
- `assets/` … 圧縮済み背景 JPEG（幅1280 / 品質80）とキャラ透過 PNG（幅600）

## 遊び方（進行フロー）

1. 導入: 「動物園に迷い込んでしまった。出口を探そう。」
2. 中央広場の看板「まっすぐ読むな」。最初はライオン未攻略だと解けない。
3. 探索: 鳥エリア・爬虫類エリアは雰囲気づくりのみ。
4. ライオンエリア（傾きチュートリアル）: 端末を左に傾ける / 肉を左へドラッグ → 肉がライオンの前へ滑り、道が開く。
5. 看板ギミック（コア）: ライオン攻略後に看板へ戻ると解けるようになる。左に55度以上4秒キープで余分な文字が滑り落ちる。
6. 残った文字＝巡回順（ライオン→とり→はちゅうるい→でぐち）。
7. 巡回: 正しい順でエリアに入ると足跡が光る。順番を間違えるとリセット。
8. 出口ゲート開放 → クリア。

## 看板の文面設計（コア謎のキモ）

看板は「夜の動物園の、少し不気味だが自然な4行の注意書き / 詩」として読める。
端末を左に傾け続けると、糊付けの弱い＝余分な文字だけが重力で滑り落ち、
各行に残るのは巡回順を示す文字列になる。「まっすぐ読むな」＝
そのまま（全文を）読むと詩に見えるが、傾けて落として残った文字こそが本当の道順、という二重構造。

| 行 | 落ちる前の全文（＝そのまま読むと自然な詩） | 落ちる文字 | 残る文字 |
|----|--------------------------------------------|-----------|---------|
| 1 | よるの**らいおん**はおきる（夜のライオンは起きる） | よ・る・の・は・お・き・る | **らいおん** |
| 2 | **とり**はそらへかえる（鳥は空へ帰る） | は・そ・ら・へ・か・え・る | **とり** |
| 3 | **はちゅうるい**はねむらない（爬虫類は眠らない） | は・ね・む・ら・な・い | **はちゅうるい** |
| 4 | そこが**でぐち**だ（そこが出口だ） | そ・こ・が・だ | **でぐち** |

残った文字を上から読むと **「らいおん / とり / はちゅうるい / でぐち」** ＝ 巡回順。

### 設計意図

- 落ちる前の4行は、それ単体で「夜の動物園で動物たちが起きている」という不気味かわいい詩として自然に読める（＝ミスリード）。
- 各行の「主語となる動物名（＋出口）」だけが糊が強く残り、述語・助詞側が滑り落ちる。残る文字列が動物名そのものなので、順番として明確に読める。
- 巡回順（ライオン→鳥→爬虫類→出口）はエリア構成と一致し、かつライオンチュートリアルで学んだ「傾ける」操作を看板で再利用させる導線になっている。

## 傾き入力

- DeviceOrientation API（`gamma` で左右傾き）。iOS 13+ は `DeviceOrientationEvent.requestPermission()` が必要なため、ユーザータップの「傾きセンサーを有効にする」ボタン経由で取得する。
- センサー不可 / PC 向け代替: 看板・肉を左へ大きくドラッグ（ポインタ操作）で傾き相当の入力になる。
- `?debug=1` でデバッグ HUD（現在の傾き角・内部カウント）を表示。キーボード左右矢印キーで傾きをシミュレートできる（ヘッドレス検証用）。

## 詰み防止

- 看板で傾き不足を一定回数繰り返すと補助テキスト「少し傾けるくらいでは足りないようだ。」を表示。
- ライオン未攻略で看板に来た場合は「今はまだ、意味が分からない。」と明示し解けない旨を伝える。

## 状態保存

- 進行状態は `localStorage`（キー `zoo-escape:v1`）に保存。リロードで継続。タイトルの「はじめから」でリセット。

## GA4 計測

`analytics.js` の `window.AkashicAnalytics.track(event, params)` を通じて送信。
`GAME.id = "zoo-escape"` / `GA4_ID = "G-F53JLZ0Z44"`。
イベント: `game_start` / `area_enter` / `meat_delivered` / `sign_solved` / `patrol_complete` / `game_clear`。
`localhost` 等では `isLive=false` となり送信せず、`?debug=1` で console にログ出力。

## 素材

`character_*.png` はオフホワイト背景を ImageMagick の floodfill で透過化。
`background_*` は sips で幅1280 / JPEG 品質80 に圧縮。元素材のファイル名は内容と入れ替わっていたため、
中央広場（「まっすぐ読むな」看板つき）を `bg_central_sign.jpg`、入口/出口ゲートを `bg_exit_gate.jpg` にリネーム済み。
